# Role Hierarchy

1. **ArchTechnomancer** (Orchestrator)  
   - Powers: planning, cost control, web.search tool, repo write access.  
2. **HighTechnomancer-<Expertise>** (High Technomancers)  
   - Powers: break milestones into epics; spawn Technomancers; review PRs.  
   - Examples: HighTechnomancer-Engineering, HighTechnomancer-QA  
3. **Technomancer-<Expertise>** (Technomancer Workers)  
   - Powers: write/modify code; run unit tests in Codex sandbox.  
   - Example: Technomancer-Code  
4. **ApprenticeTechnomancer-<Expertise>** (Apprentice Technomancers)  
   - Powers: implement small tickets; fix lint/tests; ask clarifying questions.
